# Contributor Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/), v2.1.

In short: treat everyone with respect, give helpful feedback, assume good faith, and don't be a jerk. Report unacceptable behavior to `conduct@lumen.example`. Maintainers may remove or ban contributors who violate the covenant.
